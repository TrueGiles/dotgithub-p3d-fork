# Welcome to the P3D Project!

This organizaion is where the codebase for all P3D-related projects will live!
## [Join the Discord!](https://discord.gg/CQnkc5gS6a)
On Discord, we have general discussions with the community about the game we're creating *together*.

## What's my role here?
- We recommend checking out the #roles channel and assigning yourself a role under "Consoles" on the **DS consoles you have access to**.    
- If you're interested in **helping out without joining the team**, we recommend checking out the #roles channel & adding yourself as a @Tester.
- If you're interested in **joining the team**, we recommend checking out the #roles channel & adding yourself as an @Applicant. You will then see a new #apply-to-the-team channel under "Team Application", where you can begin the application process

## So, where's the game?
